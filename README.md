# 👾 Space Invaders
Uma recriação do clássico jogo de arcade Space Invaders, desenvolvida para fins de estudo de lógica de programação, orientação a objetos, estruturas de dados e desenvolvimento de jogos.

## 📝 Sobre o Projeto

O objetivo é simples, mas desafiador: defender a Terra contra ondas de alienígenas invasores que se movem lateralmente e atiram no jogador. À medida que o jogo avança, a velocidade aumenta, testando os reflexos do jogador.

### Principais Focos de Aprendizado:
* **Game Loop:** Gestão de eventos, atualizações de estado e renderização.
* **Estruturas de Dados:**  organização, armazenamento e manipulação eficiente de informações.
* **Colisões:** Implementação de lógica de impacto entre lasers e entidades.
* **UI/UX:** Criação de menus, telas de Game Over e HUD de pontuação.

# 🕹️ Como Jogar

### Pré-requisitos
Certifique-se de ter o Python instalado. Se não tiver o Pygame, instale via terminal:
```bash
'pip install pygame'
```

## 💾 Instalação

* I. Clone o repositório:
```bash
git clone https://github.com/GuilhermeMaglia/Space_Invaders.git
```
* II. Acesse a pasta do projeto:
```bash
cd Space_Invaders
```
* III. Execute o jogo:
```bash
python game.py
```

---

## 🎮 Comandos

* **Setas Esquerda/Direita:** Movem a nave.
* **Barra de Espaço:** Dispara o laser
